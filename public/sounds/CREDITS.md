# Sound Credits

All recorded loops in this folder are derived from **public-domain (CC0 1.0)**
field recordings. Under CC0 no attribution is required, but we credit the
sources here for provenance and gratitude.

Each file was trimmed to a representative ~40-second section, loudness-normalized
(approx. −20 LUFS), and given a 4-second tail→head crossfade so it loops
seamlessly.
